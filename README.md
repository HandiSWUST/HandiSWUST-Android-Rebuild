# 掌上西科-安卓端
## 前言
本质上就是一个浏览器，但是加入了一个课程表小部件，设计上参考了ScheduleX，由于本人主攻方向为后端，Android部分的代码可能稍显混乱，如对您造成困扰本人深感抱歉

## 项目构建
在Android Studio(由于使用了7.3以上的Gradle，不能使用IDEA)中直接导入本项目即可，IDE会帮你做好一切
